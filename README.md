# Product UGC Pipeline

`product-ugc-pipeline` is a Codex Skill for ecommerce product UGC video production.

It converts product URLs into a traceable production package: product scraping, product-image analysis, product cognition, UGC prompt variants, start/end keyframes, and VEO video generation.

## Contents

- `SKILL.md`: full production workflow and quality contract.
- `SKILL_INTRO.md`: readable introduction and process overview.
- `scripts/`: production scripts for scraping, analysis, product brief generation, prompt generation, image generation, video generation, and fresh batch reruns.
- `references/`: provider and API notes.
- `docs/`: supporting documentation.

## Safety Notes

This repository is prepared as a sanitized upload package. API keys, generated outputs, caches, logs, and local runtime files are intentionally excluded.
